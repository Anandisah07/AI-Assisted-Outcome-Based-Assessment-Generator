🎓 AI-Assisted Curriculum Analysis & Outcome-Based Assessment Generator
 📖 Overview
This project demonstrates how Artificial Intelligence can assist in curriculum analysis, assessment design, and educational analytics.

Using the **Database Management System (DBMS)** curriculum, I created a structured assessment bank of **100 unique multiple-choice questions**, organized them using Microsoft Excel, and built an interactive dashboard in **Microsoft Power BI** to visualize curriculum coverage and assessment insights.

🚀 Features

- 📚 Curriculum Analysis
- 📝 100 Unique DBMS MCQs
- 🎯 Bloom's Taxonomy Classification
- 📊 Difficulty Classification (Easy, Medium, Hard)
- 📖 Learning Outcome Mapping
- 📈 Interactive Power BI Dashboard
- 🔍 Topic-wise Assessment Analysis
- 🤖 AI-Assisted Question Generation
